# HiveMind

Marketing site for **HiveMind** — shared collective memory for OpenHome voice agents.

## Stack

- React + Vite + TypeScript
- Tailwind CSS
- Framer Motion
- D3.js

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Deployed on [Vercel](https://vercel.com). Pushes to `main` trigger production deploys when the repo is connected.
